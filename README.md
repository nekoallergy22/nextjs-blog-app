# ミニマム Blog

シンプルでミニマムなブログアプリケーション。Next.js 15、TypeScript、Tailwind CSS を使用して構築されています。

## 機能

- マークダウン形式のブログ投稿の作成、編集、削除
- レスポンシブデザイン
- モダンな UI

## 起動方法

### 開発環境

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

開発サーバーは http://localhost:3000 で起動します。

### 本番環境

```bash
# 依存関係のインストール
npm install

# プロダクション用ビルド
npm run build

# プロダクションサーバーの起動
npm start
```

プロダクションサーバーは http://localhost:3000 で起動します。
