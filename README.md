# ミニマム Blog

シンプルでミニマムなブログアプリケーション。

## 起動方法

### 開発環境

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

開発サーバーは http://localhost:3000 で起動します。

### 本番環境

```bash
# 依存関係のインストール
npm install

# プロダクション用ビルド
npm run build

# プロダクションサーバーの起動
npm start
```

プロダクションサーバーは http://localhost:3000 で起動します。
